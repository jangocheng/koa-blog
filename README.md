# 在线知识分享项目 [后端项目]

> 在线知识分享项目是作者个人毕业项目。
<br>项目技术栈为: (vue2 + muse-ui) + (Node.js + Koa2 + JWT) + (MongoDB)
<br>欢迎喜欢的朋友下载使用

## 启动项目

``` bash

npm install

# serve with hot reload at localhost:3000
npm  start
```
